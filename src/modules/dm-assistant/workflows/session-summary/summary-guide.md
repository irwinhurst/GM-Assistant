# Session Summary Generation

## Log Analysis Process

### Session Review
1. **Parse session-log.md** - Extract chronological events
2. **Analyze combat logs** - Compile combat statistics and outcomes
3. **Track character changes** - HP, conditions, equipment, experience
4. **Identify story beats** - Key decisions, discoveries, plot advancement

### Summary Components

#### Character Status Updates
```markdown
## Character Updates
### [Character Name]
- **Hit Points:** [Current]/[Max] (Change: [+/-X])
- **Conditions:** [Current active conditions]
- **Experience:** [Current XP] (+[gained this session])
- **Equipment Changes:** [New items gained/lost]
- **Spell Slots:** [Current status by level]
```

#### Story Progress
```markdown
## Story Developments
### Major Events
- [Key plot developments]
- [Important discoveries]
- [Significant character decisions]

### Areas Explored
- [New locations visited]
- [Encounters completed]
- [Ongoing mysteries/hooks]

### NPCs Encountered
- [New NPCs met and their significance]
- [Relationship changes with known NPCs]
```

#### Combat Summary
```markdown
## Combat Encounters
### [Encounter Name]
- **Duration:** [Number of rounds]
- **Outcome:** [Victory/defeat/fled/negotiated]
- **Key Moments:** [Critical hits, clutch saves, creative solutions]
- **Resources Used:** [Major spell slots, items, abilities]
```

### Next Session Prep Notes
- **Immediate Continuity:** Where session ended, what happens next
- **Outstanding Hooks:** Unresolved plot threads and clues
- **Character Needs:** Healing, equipment repair, level advancement
- **DM Reminders:** NPCs to prepare, areas to detail, rules to review
