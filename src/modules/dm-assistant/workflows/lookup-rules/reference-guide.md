# Rule Reference System

## Quick Lookup Categories

### Spells
- **Search by:** Name, level, school, class
- **Display:** Components, casting time, range, duration, description
- **Example:** "fireball" → 3rd level evocation, 8d6 fire damage, 20-foot radius

### Conditions
- **Common:** Blinded, charmed, frightened, grappled, paralyzed, poisoned
- **Display:** Mechanical effects, duration, removal conditions
- **Example:** "poisoned" → Disadvantage on attack rolls and ability checks

### Monster Abilities
- **Search by:** Creature name or ability name
- **Display:** Action type, recharge, damage, save DC
- **Example:** "dragon breath" → Recharge 5-6, Dex save DC 19

### Equipment
- **Weapons:** Damage dice, properties, range
- **Armor:** AC bonus, strength requirement, stealth disadvantage
- **Magic Items:** Attunement, charges, activation

### Core Rules
- **Combat:** Attack rolls, damage, initiative, actions
- **Exploration:** Movement, vision, environmental hazards
- **Social:** Ability checks, advantage/disadvantage, proficiency

## Reference Format
```markdown
## [Rule/Spell/Condition Name]
**Type:** [Category]
**Source:** [Book and page]

**Quick Summary:** [One-line description]

**Mechanics:**
- [Key mechanical effects]
- [Duration/range/components if applicable]
- [Save or check requirements]

**Table Note:** [Any important timing or interaction notes]
```

## No Session Logging
Rule lookups are NOT logged to session notes - they're quick references that don't affect game state.
