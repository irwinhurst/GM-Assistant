# Import Review Checklist

## Content Validation

### Area Files
- [ ] **All areas processed**: Verify expected number of area files created
- [ ] **Consistent formatting**: All files follow area template structure
- [ ] **Complete descriptions**: No missing or truncated area descriptions
- [ ] **Page references**: All areas include original page numbers
- [ ] **Map keys**: Area identifiers match source material

### Area Content Quality
- [ ] **Readable descriptions**: Text extracted accurately from source
- [ ] **Key features identified**: Important elements properly highlighted
- [ ] **Hidden elements noted**: Secret doors, traps include proper DCs
- [ ] **Treasure documented**: Loot and important items captured
- [ ] **Special rules included**: Unique mechanics or environmental effects

### Navigation Links
- [ ] **All connections present**: Doors, passages, stairs properly linked
- [ ] **Bi-directional linking**: Connections work both ways
- [ ] **Link syntax correct**: Markdown links properly formatted
- [ ] **File paths valid**: All linked files exist
- [ ] **Requirements noted**: Access conditions clearly specified

### Encounter Integration
- [ ] **Combat encounters extracted**: Fight scenes properly documented
- [ ] **Trap encounters noted**: Hazards and traps captured
- [ ] **NPC interactions**: Social encounters identified
- [ ] **Encounter links functional**: Links from areas to encounter files work
- [ ] **Trigger conditions clear**: When encounters activate

## Common Import Issues

### Text Extraction Problems
- **Garbled text**: OCR errors in scanned content
- **Missing sections**: Incomplete extraction from complex layouts
- **Merged descriptions**: Multiple areas combined incorrectly
- **Split content**: Single area broken across multiple files

### Connection Mapping Errors
- **Missing connections**: Doors or passages not identified
- **Incorrect directions**: North/south orientation errors
- **Broken links**: File name mismatches or typos
- **One-way links**: Missing reverse connections

### Encounter Extraction Issues
- **Missed encounters**: Combat or traps not detected
- **Incomplete stat blocks**: Missing creature information
- **Wrong locations**: Encounters assigned to incorrect areas
- **Trigger confusion**: Unclear activation conditions

## Review Process

### 1. Quick Scan
- **File count**: Does it match expected areas?
- **File names**: Consistent naming convention used?
- **Template structure**: All files follow standard format?
- **Link format**: Navigation links properly formatted?

### 2. Content Spot Check
- **Random area selection**: Pick 3-5 areas to review in detail
- **Compare to source**: Verify against original adventure text
- **Test navigation**: Click through several area connections
- **Check encounters**: Verify encounter files load correctly

### 3. Navigation Testing
- **Start from entry point**: Begin at adventure starting area
- **Follow logical path**: Trace expected player route
- **Test secret connections**: Verify hidden passages work
- **Check dead ends**: Ensure they're intentional

### 4. Encounter Verification
- **Combat readiness**: Stat blocks complete and accurate
- **Trap functionality**: DCs and effects properly noted
- **NPC information**: Social encounters have sufficient detail
- **Treasure tracking**: Rewards properly documented

## Correction Guidelines

### When to Use Update-Area Workflow
- **Text accuracy issues**: Wrong or garbled descriptions
- **Connection problems**: Missing or incorrect links
- **Encounter errors**: Missing or misplaced encounters
- **Format inconsistencies**: Template structure problems

### When to Re-Import
- **Major extraction failures**: Large sections missing or corrupted
- **Wrong source material**: Incorrect adventure content processed
- **System errors**: Technical problems during processing
- **Fundamental structure issues**: Core organization problems

### Quick Fixes
- **Typos and small errors**: Minor text corrections
- **Link fixes**: Correcting file name mismatches
- **Missing connections**: Adding overlooked doors/passages
- **Format adjustments**: Template structure corrections

## Sign-Off Criteria

### Ready for Table Use
- [ ] All expected areas present and properly formatted
- [ ] Navigation links tested and functional
- [ ] Encounters extracted and accessible
- [ ] No major content gaps or errors
- [ ] DM comfortable with import quality

### Needs Additional Work
- [ ] Significant text extraction errors
- [ ] Missing or broken navigation
- [ ] Important encounters not captured
- [ ] Major format inconsistencies
- [ ] DM requests improvements

## Next Steps After Review

### If Import Successful
1. **Generate table notes** using table-notes workflow
2. **Test run navigation** during actual prep
3. **Bookmark frequently used areas** for quick access
4. **Prepare encounter materials** for likely combats

### If Import Needs Work
1. **Document specific issues** for targeted fixes
2. **Use update-area workflow** for individual corrections
3. **Consider partial re-import** for severely broken sections
4. **Prioritize fixes** based on session timeline
