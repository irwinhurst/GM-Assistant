# Generate Dungeon Workflow

This workflow guides you through creating a complete 5-room dungeon using proven design methodology, scalable encounters, and session-ready files for DM Assistant.

## Purpose
- Design and generate a 5-room dungeon for D&D sessions
- Scale encounters for party level and size
- Prepare all area and encounter files for session play

## Steps
1. **Define Dungeon Parameters**
   - Set party level, theme, and requirements
2. **Choose Dungeon Theme**
   - Select thematic elements and style
3. **Generate 5-Room Structure**
   - Create each room using the 5-room methodology
4. **Scale Encounters**
   - Adjust difficulty for party level and size
5. **Create Dungeon Map**
   - Design spatial layout and room connections
6. **Prepare Session Files**
   - Generate area and encounter files for session runner

## Outputs
- Dungeon folder and index
- Area files for each room
- Encounter files for each challenge
- Complete dungeon summary for session play

## Integration
- Compatible with session-prep, session-runner, and campaign-manager agents
- Works with create-area and update-area workflows
- Supports session folder structure and navigation

## Best Practices
- Use the 5-room methodology for balanced dungeon design
- Scale encounters using SlyFlourish benchmarks
- Validate all area and encounter files before play
- Maintain clear navigation and session integration
